CNN vs RNN

CNN (Convolutional Neural Network):
- Strong for spatial data such as images.
- Uses convolution and pooling to capture local patterns.
- Common uses: image classification, object detection.

RNN (Recurrent Neural Network):
- Designed for sequential data such as text or time series.
- Keeps a hidden state to represent previous context.
- Variants like LSTM and GRU improve long-term memory handling.

Simple rule of thumb:
Use CNN for 2D spatial structure; use RNN/LSTM/GRU for ordered sequences.
